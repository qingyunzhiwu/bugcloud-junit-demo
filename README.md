# BugCloud-JUnit-Demo

#### 介绍

BugCloud Junit Demo 是 BugCloud Junit Core 库的测试示例工程，用于展示如何快速创建自动化单元测试，如何通过配置参数使测试报告结果自动推送到[ButCloud云平台](http://www.bug-cloud.com)。
本示例采用SpringBoot架构，代码中实现了一个Controller接口类，用于测试调用。
